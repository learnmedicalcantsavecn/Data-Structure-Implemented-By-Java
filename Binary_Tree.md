### Binary Tree

```java
/* 二叉树遍历框架 */
void traverse(TreeNode root) {
    // 前序遍历
    traverse(root.left)
    // 中序遍历
    traverse(root.right)
    // 后序遍历
}
```

**快速排序就是个二叉树的前序遍历, 归并排序就是个二叉树的后序遍历**.

### 翻转二叉树

```
     4               4
   /   \            /  \
  2     7      ->  7    2
 / \   / \        / \  / \
1   3 6   9      6   9 1  3
```

我们发现只要把二叉树上的每一个节点的左右子节点进行交换, 最后的结果就是完全翻转之后的二叉树.
